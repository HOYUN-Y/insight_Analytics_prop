insightAnalyticsProp

의미 & 컨셉

insight Analytics Prop는 분양대행사 마케팅 기획 담당자가 설치 없이 브라우저에서 부동산 데이터를 수집·정제·분석·시각화하고 시장조사 보고서·분양 제안서까지 한 흐름으로 만드는 로컬-퍼스트 부동산 분석 워크벤치입니다.

insight Analytics의 부동산(Property) 에디션으로, 형제 제품의 sight 주황 헤리티지와 The Insight Point 로고마크를 그대로 계승합니다. 여기에 부동산 도메인을 뜻하는 Prop을 연두색으로 더해 에디션을 구분합니다. 주황은 제품군 헤리티지(고정), 연두는 이 에디션의 UI accent 색입니다.

로고마크

The Insight Point — 축(axis) 위로 상승하는 데이터 궤적이 하나의 초점(focal point)으로 수렴하는 형태. 분석(Analytics)과 통찰(sight)을 한 글자에 담습니다. 심볼은 제품군 공통 자산이므로 주황 헤리티지를 그대로 유지합니다 — 라운드 스퀘어(반경 17/72 ≈ 23%) 배경에 흰색 3.4pt 스트로크, 마지막 점만 채워 "발견의 순간"을 강조합니다. 파비콘·앱 아이콘·레일 마크에 그대로 사용합니다.
권장
  • 심볼은 주황 헤리티지 고정 (제품군 공통)
  • 제품 구분은 워드마크의 Prop(연두)으로
  • 최소 여백 = 마크 높이의 25% 유지
지양
  • 심볼을 연두/타 색으로 바꿔 변형
  • "sight" 주황을 경고·오류 의미로 사용
  • 궤적·초점 점 제거하거나 회전
